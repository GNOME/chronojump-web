Chronojump software on OSX El Capitan

Chronojump is not easy to install right now on in the new IOS: El Capitan. The only solution we found at the moment is written on this page. Obviously we hope to find a better solution soon and then publish here how to revert the changes you made.

  Installation instructions (do this only one time)

  1. Disable csrutil
    • Boot to Recovery OS: You need to restart your computer holding down the Command and R keys (⌘+R) at startup.
    • Launch Terminal from the Utilities (or Tools) menu.
    • Enter this command: csrutil disable↩
    • Reboot the machine
  2. Deactivate Gatekeeper
  3. Install the last version of R: Latest R version for MacOSX
  4. Install the newest Chronopic driver
  5. Install Chronojump 1.5.4. On the end of the installation it will ask if you want to open it, DO NOT OPEN IT because it will not work.
  6. Create a link to Chronojump
    • Open Finder, press the keys: ⌘+⇧+G
    • Write this: /Applications/Chronojump.app/Contents/MacOS/↩
    • Select the file "Chronojump" and "Make alias" or press ⌘+L
    • An "Chronojump.alias" file will be created. Move the new file to the Desktop.

  Execute Chronojump

Open the "Chronojump.alias" file on your Desktop.
If you have any problem write to support@chronojump.org
